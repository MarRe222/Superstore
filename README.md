# Superstore Analytics Project — Work in Progress

This repository contains an ongoing analytics project built around the Superstore dataset.  
The current focus is on establishing a solid foundation through data cleaning and exploratory analysis.

### Planned Components
- A reusable data cleaning pipeline (`clean.py`)
- Feature engineering pipeline (`features.py`)
- A structured EDA notebook with visual insights (`01_data_exploration.ipynb`)
- A dashboard displaying core KPIs (sales, profit, margins, segments)
- Customer and product performance analysis
- Final documentation and project summary

The project is actively being developed and expanded.
